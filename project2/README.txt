# Knapsack 0-1 Dynamic Program

Steps for the User:
0. Run createInputFile.cpp to create a file (or not!)
1. Run driver.cpp
2. Enter filename.txt (that you made using createInputFile.cpp or some other method)

What should happen:
***Program should output the 0/1 Knapsack Problem solution using the dynamic programming approach
***Program should output the 0/1 Knapsack Problem solution using the refinement of the dynamic programming approach
